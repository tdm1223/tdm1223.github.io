## [9095. 1, 2, 3 더하기](https://www.acmicpc.net/problem/9095)
### 문제 요약
- 다이나믹 프로그래밍 문제
- 정수 n을 1, 2, 3의 조합으로 나타내는 방법의 수를 구하는 문제이다.

### 풀이 과정
```
n=4이면
1+1+1+1
1+1+2
1+2+1
2+1+1
2+2
1+3
3+1
=> 총 7가지가 된다.
```

- D[n] : n을 1, 2, 3의 조합으로 나타내는 방법의 수
1. D[n-1]에서 1 더하기
2. D[n-2]에서 2 더하기
3. D[n-3]에서 3 더하기
- D[n]은 D[n-1]+D[n-2]+D[n-3]으로 표현할 수 있다.


### 소스 코드
```cpp
#include<iostream>

using namespace std;

int d[11]; // d[i] : 1,2,3의 합으로 i를 만들 수 있는 경우의 수
int main()
{
    int n, k;
    cin >> n;
    while (n > 0)
    {
        n--;
        cin >> k;
        d[0] = 1;
        d[1] = 1;
        d[2] = 2;
        for (int i = 3; i <= k; i++)
        {
            d[i] = d[i - 1] + d[i - 2] + d[i - 3];
        }
        cout << d[k] << endl;
    }
    return 0;
}
```
